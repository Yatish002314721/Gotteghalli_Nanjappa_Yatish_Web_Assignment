###Portfolio Website

*****************************Favicon*****************************
<link rel="icon" href="favicon.ico" type="image/x-icon">
A small icon displayed in the browser tab.
Table:

*****************************<table>...</table>*****************************
Used to organize data in a structured format with rows and columns.
Form:

*****************************<form>...</form>*****************************
A container for input elements that collects user data.
Images:

*****************************<img src="image.jpg" alt="Description">*****************************
Displays images on the webpage.
Hyperlink:

*****************************<a href="https://example.com">Link Text</a>*****************************
A clickable link that directs users to another page or resource.
Button:

*****************************<button>Click Me</button>*****************************
A clickable button element that can trigger actions or submit forms.
Audio:

*****************************<audio controls><source src="audio.mp3" type="audio/mpeg"></audio>*****************************
Embeds audio content with controls for play, pause, etc.
Video:

*****************************<video controls><source src="video.mp4" type="video/mp4"></video>*****************************
Embeds video content with playback controls.
Header:

*****************************<header>...</header>*****************************
Represents introductory content or navigational links.
Footer:

*****************************<footer>...</footer>*****************************
Contains footer information, such as copyright and links.
Summary:

*****************************<summary>...</summary>*****************************
A summary of the content for <details> elements, which can be expanded or collapsed.
Menu:

*****************************<nav>...</nav>*****************************
Defines a set of navigation links.
Tel for Contact Information:

*****************************<a href="tel:+1234567890">Call Us</a>*****************************
A link that allows users to call a phone number directly.

*****************************Mailto for Contact Information*****************************
<a href="mailto:example@example.com">Email Us</a>
A link that opens the default email client to send an email.
Hover Effects:

CSS styles applied to elements on mouse hover to enhance interactivity (e.g., changing scale).

*****************************Figure Captions*****************************
<figure><img src="image.jpg" alt="Description"><figcaption>Image caption here</figcaption></figure>
A container for images or illustrations with a caption.

*****************************Responsive Design*****************************
Media queries in CSS to adjust styles for different screen sizes.
